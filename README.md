# 🍳 Cooking Recipes Project - Документация

Това е официалната документация за контейнеризирания проект **"Уебсайт за готварски рецепти"**. Приложението използва трислойна архитектура, разделена на независими услуги.

---

## 🚀 1. Изграждане и стартиране на контейнерите

## Изпълнение в Docker Terminal (Linux / WSL)

1. Отворете вашия терминал.
2. Навигирайте до главната папка на проекта:
   ```bash
   cd /mnt/c/Users/user/Desktop/site_with_recipes
   
3.Изградете образите и стартирайте контейнерите в реално време:

   ```bash
   sudo docker compose up --build

Тази команда автоматично ще свали базовите среди (Nginx, MySQL), ще изпълни инструкциите от Dockerfile за PHP бекенда, ще конфигурира мрежата и ще стартира услугите напълно автоматично.

4.Достъп до уебсайта:
След като контейнерите се заредят изцяло, отворете браузъра си на адрес:
👉 http://localhost:8080/login.html

5.Спиране на контейнерите:
За да изключите приложението и да освободите използваните портове, натиснете Ctrl + C в терминала или изпълнете:

   ```bash
   sudo docker compose down
